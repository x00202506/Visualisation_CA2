Bootstrap Features Used

A technical breakdown of the Bootstrap components, utilities, and features used in my one-page interactive portfolio website.

Hero Section

  • Grid system: .container, .row, .col-md-*, .col-lg-* used to create a responsive two-column layout for my introduction text and profile image.
  • Buttons: Primary and outline buttons (.btn, .btn-outline-light, .btn-lg) used for the “Contact” and “View projects” calls-to-action.
  • Images: Responsive hero portrait styled using .img-fluid principles with a custom circular frame.
  • Utility classes: Spacing and typography utilities such as .mt-*, .mb-*, .fw-bold, and .text-muted used to control spacing and text hierarchy.

About Me Section

  • Cards: .card and .card-body used to present the About Me content in a structured panel.
  • Grid & layout: .container and .row used to align the card within the page.
  • Utility classes: Headings and supporting text styled with .mb-*, .lead, and custom section title classes.

Education Section

  • Cards: Education details displayed inside a .card with .card-body for a clear container.
  • Tables: GPA performance by year presented using .table and .table-bordered to structure rows and columns.
  • Utility classes: .mb-4, .mt-2, and .small used to format spacing and footnote text.

Work Experience Section

  • Cards: Each role (ICT End User Team, volunteer etc.) is displayed inside a .card with headings and bullet-point responsibilities.
  • Grid system: .row and .col-lg-6 create a responsive two-column layout for the different positions.
  • Lists: .list-unstyled used to present concise responsibility bullets without default browser bullets.

Projects Section

  • Cards: Project information presented using .card, .card-body, and .shadow-sm for separation and depth.
  • Buttons & links: Project buttons use .btn and .btn-primary for consistent styling with other interactive elements.
  • Grid & layout: .row and .col-lg-* manage the primary project card and supporting side content.

Technical Skills Section

  • Grid system: Two-column layout for skill headings and descriptions built with .row and .col-md-6.
  • Badges / tags: Custom skill tags styled similarly to Bootstrap .badge components to highlight technologies (SQL, Linux, Networking, etc.).
  • Utility classes: Spacing and typography via .mb-*, .fw-bold, .text-muted.

Languages Section

  • Layout utilities: .d-flex and .justify-content-between used to position language names and proficiency labels on a single line.
  • Custom progress bars: Language proficiency visualised using custom .lang-bar and .lang-bar-fill elements, inspired by Bootstrap’s .progress and .progress-bar pattern.
  • Utility classes: .small and .text-muted used for the proficiency descriptions.

Hobbies & Interests Section

  • Lists: Hobbies displayed using a clean unordered list with .list-unstyled.
  • Utility classes: Spacing controlled using .mb-* to keep the section consistent with the rest of the layout.

Contact Section & Modal

  • Modal component: Bootstrap .modal, .modal-dialog, .modal-content, .modal-header, .modal-body, and .btn-close are used to display a contact form in a dialog.
  • Form controls: Contact form fields styled with .form-control and .form-label for name, email and message inputs.
  • Buttons: The “Open contact form” trigger uses .btn and .btn-primary; the form submit button uses .btn styling.
  • Utilities: .mb-3, .small, and .text-muted used for spacing and helper text.

Navbar, Footer & Global Utilities

  • Navbar component: Responsive navigation implemented with .navbar, .navbar-expand-lg, .navbar-brand, .navbar-nav, .nav-item, .nav-link, .navbar-toggler, and .navbar-collapse.
  • Collapse behaviour: Mobile menu uses Bootstrap’s collapse JavaScript via data-bs-toggle="collapse" and data-bs-target.
  • Utility classes: Global spacing, alignment, and typography controlled using .mt-*, .mb-*, .px-*, .py-*, .ms-auto, .text-center, and .fw-bold.
  • Icons: Bootstrap Icons such as bi-sun-fill, bi-moon-fill, and bi-arrow-up used for the theme toggle and scroll-to-top button.

Custom JavaScript Functionality

In addition to Bootstrap’s built-in behaviour, I implemented custom JavaScript to satisfy the “dynamic content using JavaScript” requirement:

  • Dark mode toggle: Toggles a .dark-mode class on the <body>, switches the sun/moon icon, and saves the preference using localStorage.
  • Scroll-to-top button: Shows and hides a floating button based on scroll position and scrolls smoothly to the top of the page when clicked.
  • Animated language bars: Uses data-level attributes on language bars to animate their widths when the section comes into view.
