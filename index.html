Projects

Selected academic work demonstrating technical ability and teamwork.

2nd Year Project: Prison System Database with Live Tracking

Web application for managing prisoners, guards, cells, events and job assignments, with real-time location updates.

Collaborated on a dynamic web application designed to enhance prison management through a comprehensive database. The system stores key information about prisoners, guards, cells, events and job assignments.

A real-time events table logs timestamps and locations for each prisoner, updated using the MQTT messaging protocol to ensure accurate tracking. A user-friendly interface enables seamless interaction with the database, while an “About” page highlights each team member’s role and contributions.

Tools & Technologies
  • Application Development: Visual Studio Code
  • Front-end: HTML, CSS, JavaScript, Bootstrap
  • Database: Supabase (SQL) for data storage and management
  • Messaging: MQTT with Google Cloud Shell for real-time updates
  • Collaboration: Azure DevOps for planning, communication and version control

This project strengthened my skills in full-stack development, database design and cloud services, and underlined the value of teamwork and structured project management on complex systems.
